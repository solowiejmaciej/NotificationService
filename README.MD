
# CheapFreak

## Features

- Docker
- Microservices
- Background jobs
- Cache
- RabbitMq
- JWT Auth
- Refresh Tokens
- Email sending with SMTP
- Push sending with Firebase
- SMS sending with SMSPlanetAPI
